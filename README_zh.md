## 🔭 目前專案

### 🚁 Vision-DPPO
> 端對端無人機控制 — 透過擴散策略 (Diffusion Policy)

設計端對端視覺運動控制框架，以擴散策略取代串級PID控制器，將原始FPV影像序列直接映射至4D馬達推力，透過CNN編碼器與條件式1D U-Net實現。

- 🛠️ **技術棧**: Python, PyTorch, ROS 2, CUDA
- 🚩 **狀態**: 進行中的研究 (2026年2月 ~ 至今)
- 🎯 **目標**: 原型化核心軟體基礎設施，包含自訂6自由度四旋翼模擬（RK4積分@200Hz）、基於狀態的PPO專家以及HDF5合成數據收集管線

### 🍽️ 食來運轉 (FoodFate)
> 智能隨機餐廳推薦 App

解決"今天吃什麼"的難題！透過個性化篩選、趣味輪盤和一站式行動功能，幫助用戶快速決定用餐地點。

- 🛠️ **技術棧**: Flutter, Python Flask, PostgreSQL, Google Maps API
- 🚩 **狀態**: 積極開發中 (MVP階段)
- 📋 [專案詳情](https://github.com/Ben0126/food_fate)

### 🏭 工廠 ERP 系統

為製造業設計的訂單處理和生產管理系統，包含客戶管理、產品追蹤和原料採購等功能。旨在數位化製造業工作流程，取代傳統的Excel操作。
